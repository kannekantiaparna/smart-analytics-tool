# 🛠️ Smart Analytics Tool

A smart CSV analytics tool that automatically analyzes
any uploaded dataset and generates dynamic visualizations.

## ⚙️ Features
- CSV Upload Feature
- Dataset Preview (shape, columns, data types)
- Missing Value Analysis with visualization
- Basic Statistical Summary
- 4 Dynamic Visualizations:
  - Distribution of numerical columns
  - Top categories bar chart
  - Correlation heatmap
  - Category pie chart

## 🛠️ Tech Stack
Python | Pandas | NumPy | Plotly | Google Colab

## 📁 Sample Dataset
[Netflix Movies and TV Shows – Kaggle](https://www.kaggle.com/datasets/shivamb/netflix-shows)

## ▶️ How to Run
1. Open `smart_analytics_tool.ipynb` in Google Colab
2. Run all cells in order
3. Upload any CSV file when prompted
4. View auto-generated analysis and visualizations
